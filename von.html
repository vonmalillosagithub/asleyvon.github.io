HTML Markup Tags

Tag Description Possible Attributes Attribute Description
<html> Defines the root of an HTML document lang Specifies the language of the document
<head> Contains meta-information about the HTML document
<title> Defines the title of the HTML document
<body> Contains the content of the HTML document bgcolor,text bgcolor,textcolor
<h1> to <h6> Defines header levels from 1 to 6 align,class,id align,class,id
<p> Defines a paragraph align align
<a> Defines a hyperlink href, target Specifies the URL and target of the hyperlink
<img> Embeds an image src, alt, width, height Specifies the image source, alternate text, width, and height
<ul> Defines an unordered list type,class,id type,class,id
<ol> Defines an ordered list start,type,class,id start,type,class,id
<li> Defines a list item value,class,id value,class,id
<br> Defines a line break
<hr> Defines a horizontal line
<div> Defines a division or section in an HTML document class,id,styles class,id,styles (CSS Style)
<span> Defines an inline container for text class,id styling/scripting
<b> Defines strong importance text bold text
<em> Defines emphasized text class,style styling,functionality
<tr> Defines table row align,bgcolor align,bgcolor
<form> Difines HTML Form action,method action,method
<table> creates table border,width,cellpadding,cellspacing border,width,cellpadding,cellspacing
